== Description ==
Displays featured product first everywhere with any sorting order. It provides WooCommerce Featured Product sidebar widget.  It is fully compatible to woocommerce.

Maximize your profitability by putting your favourite product fist. 

#### Full Features List:
* Displays Feature Product first On any sorting order
* Featured Product First on Shop, Archive and Product Search Page
* Displays featured product first in custom wp query
* NO LIMITATIONS on the number of featured products displayed first.
* Featured Product Widget
* Configurable

== Installation ==
1.Download your WordPress Plugin to your desktop. If downloaded as a zip archive, extract the Plugin folder to your desktop.
2.Read through the "readme" file thoroughly to ensure you follow the installation instructions.
3.With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
4.Go to Plugins screen and find the newly uploaded Plugin in the list.
5.Click Activate to activate it.
6.Go to WooCommerce->Settings and navigate to products tab and then display section and set settings related to featured product first


== Frequently Asked Questions ==

= How can i set featured product? =
To know how to set featured product in WooCommerce, Please refer official documentation [https://docs.woocommerce.com/document/managing-products/#section-10](https://docs.woocommerce.com/document/managing-products/#section-10)

= From where i can make setting of this plugin? =
Please go to WooCommerce > Settings > Products.

= Is this plugin WooCommerce compatible? = 
Yes, absolutely.

= Is this plugin compatible with WooCommerce 3.x? =
Ans.Yes, absolutely.

=  Is this plugin compatibe with WooCommerce older version like 2.x compatible? = 
The free version of plugin is compatible to  WooCommerce older version like 2.x, but premium version is not compatible to  WooCommerce older version like 2.x.

= Where can i set default catalog product default sorting order? =
To set default catlog product default sorting order, Please go to Admin Dashboard > Appearance > Customize > WooCommerce > Product Catalog. Here, There are second field where you can choose "Default product sorting"


= How can i buy premium version of plugin? =
Go to your site Admin Dashboard > Featured Product First > Upgrade

= What is main diffrence between free verion and premium version of plugin? =
Free version of plugin displays featured product first on shop, archive and search page with woocommerce default sorting. If you would like to sorting everyewhere, Please Premium version of plugin.

= Why i can't see setting displayed as in screenshot? =
The screenshots are of premium version, To unlock full featues, Please buy pro version.
